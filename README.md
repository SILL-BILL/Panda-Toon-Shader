# Panda Toon Shader

Panda Toon Shader is an anime-style toon shader add-on for Blender.

This project aims to provide a lightweight and flexible toon shading workflow for stylized characters, combining simple shadows with expressive highlights and rim lighting.

The shader is currently under development and research, starting from a node-based material designed for Blender 5.1 and planned to be distributed as a Blender add-on.

---

## Features

- Toon Shadow with ColorRamp
- Custom Shadow Color
- Rim Light
- Toon Metal Highlight
- Fur Shading Support
- Alpha Clip / Transparency Support
- Emission Support

## Philosophy

Panda Toon Shader does not aim for physically accurate rendering.

Instead, it focuses on:

- Clear toon shadows
- Appealing highlights
- Anime-style materials
- Lightweight workflows
- Easy customization

Create expressive stylized characters with simple and powerful toon shading.
